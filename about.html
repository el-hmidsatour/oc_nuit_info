// PROJECT_OVERVIEW

The CYBER_ORB is a WebGL-based audio analysis tool rendered entirely in ASCII characters. It utilizes the Three.js library to calculate 3D geometry and maps luminance to text characters in real-time.

// TECHNICAL_SPECS

> Library: Three.js r128
> Effect: AsciiEffect.js
> Audio: Web Audio API (FFT Analysis)
> Animation: Vertex Displacement (Liquid Sine Wave)

Upload any MP3 or WAV file to see the liquid core react to frequency bands. The liquid motion algorithm ensures the visual remains alive even during silence.
